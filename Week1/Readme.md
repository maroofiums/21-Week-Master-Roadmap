# **Week 1 Plan: Python & Data Foundations (Day-by-Day)**

**Goal:** Build a **solid Python + data handling foundation** that will support ML, DL, CV, NLP, and pipelines.

**Daily Time:** ~3–4 hours

---

## **Day 1: Python Basics**

**Topics:**

* Variables, data types (int, float, str, list, tuple, dict, set)
* Operators: arithmetic, comparison, logical
* Conditional statements: if, else, elif

**Practice:**

* Write small programs:

  * Check if a number is prime
  * FizzBuzz problem
  * Calculate factorial

**Tip:** Focus on **writing clean Python code**; avoid copy-pasting solutions.

---

## **Day 2: Loops & Functions**

**Topics:**

* Loops: for, while
* List comprehensions & dictionary comprehensions
* Functions: def, return, parameters, default arguments

**Practice:**

* Create functions to:

  * Sum all even numbers in a list
  * Count vowels in a string
  * Fibonacci sequence generator

**Tip:** Functions will later help you build **clean ML/DL pipelines**.

---

## **Day 3: Python OOP**

**Topics:**

* Classes and objects
* Attributes & methods
* `__init__` constructor
* Inheritance basics

**Practice:**

* Create a `Student` class with attributes: name, age, marks
* Add method to calculate grade
* Create multiple student objects and print their details

**Tip:** OOP is very important for **PyTorch models, CV pipelines, and reusable ML code**.

---

## **Day 4: Python Advanced Features**

**Topics:**

* Lambda functions
* map(), filter(), reduce()
* Decorators (basic idea)
* Exception handling: try/except

**Practice:**

* Filter all even numbers from a list using `filter()`
* Map a function to square all numbers in a list
* Create a decorator to time a function execution

**Tip:** These features make Python **compact and production-ready**, which you’ll use in pipelines later.

---

## **Day 5: NumPy Basics**

**Topics:**

* NumPy arrays & operations
* Array slicing & indexing
* Mathematical operations: sum, mean, dot product, transpose
* Random number generation

**Practice:**

* Create a 2D array and compute row/column sums
* Implement matrix multiplication manually & with NumPy
* Generate random dataset for practice

**Tip:** NumPy is the **backbone of all ML/DL numeric computations**.

---

## **Day 6: Pandas Basics**

**Topics:**

* Series & DataFrame creation
* Indexing, selection & filtering
* Handling missing values
* Basic aggregation: sum(), mean(), groupby()

**Practice:**

* Load a CSV dataset (Titanic or any small dataset)
* Explore: head(), tail(), shape, info(), describe()
* Handle missing values & filter rows

**Tip:** Pandas is essential for **data cleaning and preprocessing** before ML.

---

## **Day 7: Data Visualization**

**Topics:**

* Matplotlib: line, bar, scatter plots
* Seaborn: distribution plots, boxplots, heatmaps
* Customizing plots: title, labels, color, size

**Practice:**

* Visualize Titanic dataset:

  * Count of survivors vs. non-survivors
  * Age distribution (histogram)
  * Correlation heatmap of numerical features

**Mini-Project:**

* Create **a mini EDA report** for any dataset:

  * 1–2 plots per feature
  * Highlight key observations

**Tip:** Always visualize before modeling; EDA helps **spot trends, outliers, and correlations**.
